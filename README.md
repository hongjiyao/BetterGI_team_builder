# 原神 BetterGI 万能配队工具

基于 BetterGI 战斗配队逻辑的可视化配队工具，支持一键生成配队方案和导出 TXT 文件。

## 功能特点

- 🎮 直观的 Web 界面，通过下拉菜单选择角色
- ⚡ 一键生成配队方案，格式简洁规范
- 📄 自动导出 TXT 文件，文件名由角色首字组合而成
- 🛡️ 完整的角色库，包含盾、副C、减抗等分类
- 🎯 基于 BetterGI 战斗配队策略

## 配队逻辑

**标准配队公式**：盾（刚需） + 副C/后台 + 副C/后台 + 减抗/聚怪

### 角色分类

- **盾角色（7个）**：茜特菈莉、伊涅芙、钟离、莱依拉、绮良良、托马、蓝砚
- **后台/副C（28个）**：玛薇卡、杜林、迪希雅、香菱、仆人、夜兰、哥伦比娅、那维莱特、纳西妲、艾梅莉埃、丝柯克、芙宁娜、爱诺、夏洛蒂、菈乌玛、白术、珊瑚宫心海、芭芭拉、希格雯、爱可菲、菲谢尔、欧洛伦、雷电将军、久岐忍、瓦雷莎
- **减抗/聚怪（4个）**：希诺宁、枫原万叶、砂糖、琴

## 使用方法

### 开发环境运行

1. 安装依赖：
```bash
pip install gradio
```

2. 运行程序：
```bash
python team_builder.py
```

3. 在浏览器中打开显示的链接即可使用

### 可执行文件使用


1. 双击运行 `GITeamBuilder.exe`
2. 程序会自动在浏览器中打开配队工具
